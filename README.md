# 🤖 Simulador de Enjambre de Robots

Proyecto de Inteligencia Artificial y Robótica que simula un **enjambre de robots** para búsqueda y rescate. Los robots colaboran, se comunican y optimizan la cobertura del área de manera colectiva.

## 🚀 Tecnologías
- Python 3.10+
- Clases y Programación Orientada a Objetos
- Algoritmos de coordinación y comunicación en enjambres

## 📌 Funcionalidades
- **Coordinación:** Múltiples robots colaboran para cubrir un área.
- **Comunicación:** Los robots envían mensajes para compartir información sobre celdas exploradas y hallazgos.
- **Estrategia colectiva:** Algoritmos de planificación de rutas optimizan la búsqueda evitando solapamientos.

---

## ▶️ Ejecutar simulación

1. Clonar repositorio o descargar el código.
2. Asegurarse de tener Python 3 instalado.
3. Ejecutar el simulador:

```bash
python simulador_enjambre.py

```
